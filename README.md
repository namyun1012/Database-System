# Database-System 📦

이 프로젝트는 **Python**과 **PostgreSQL**을 사용하여, 사용자로부터 입력받은 명령어에 따라 데이터베이스를 조회하거나 조작하는 간단한 데이터베이스 시스템입니다.

## 📌 프로젝트 소개

- 사용자가 입력한 명령어를 파싱하여 해당하는 SQL 쿼리를 실행
- PostgreSQL 데이터베이스와 연동하여 결과를 출력
- 명령어 기반의 인터페이스

## 💡 주요 기능

- 정해진 명령어 입력 시 쿼리 실행
- PostgreSQL과의 연결 및 트랜잭션 처리

## 🛠 사용 기술

- **Language**: Python 3
- **Database**: PostgreSQL
- **Library**: psycopg2
